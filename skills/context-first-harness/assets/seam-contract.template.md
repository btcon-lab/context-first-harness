# {Seam 이름} (정본 · seam)

> **seam-only.** 이 파일은 여러 에이전트가 공유하는 **{이 seam이 공유하는 사실 종류}**만 담는다. 구현 상세({이 seam의 how — 예: 타입·인덱스·강제 코드})는 `docs/design/`에 두고 여기를 링크한다(재기술 금지).

{owner-agent} 소유. {consumer-agent들}이 참조.

## 규칙 등급과 ID

모든 계약 항목에 **등급**과 **Rule ID**를 붙인다. 등급이 없으면 모든 문장이 같은 무게가 되어 QA가 심각도를 매길 수 없고, ID가 없으면 위반 이력이 쌓이지 않는다.

| 등급 | 표기 | override | 위반 시 | ID 형식 |
|------|------|----------|---------|---------|
| **정책** | `[MUST]` | 불가 | **FAIL** — 즉시 수정 | `{SEAM}-P###` |
| **지침** | `[SHOULD]` | 가능(이유를 `open-decisions.md`에 명시) | **WARN** — 수정 권고 | `{SEAM}-G###` |
| **구현** | 등급 없음 | 자유 | 판정 대상 아님 | — |

> ID는 한 번 부여하면 **재사용하지 않는다.** 규칙이 삭제돼도 번호를 비워두고 다음 번호를 쓴다. 과거 위반 이력이 어떤 규칙에 대한 것이었는지 추적하기 위해서다.
