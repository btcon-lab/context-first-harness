## 하네스 운영 결정
_옵션 기능을 "조용히 스킵"하지 않기 위한 결정 기록. 오케스트레이터 Phase 0 게이트가 매 실행 이 표를 읽는다._

**상태 값:** `연기`(조건 충족 시 실행 예정 — 게이트가 매 실행 재평가) · `설치됨`(완료 — 게이트가 건너뜀) · `생략`(안 함 — 게이트가 건너뜀) · `미설치`(현재 불필요, 재검토 조건 있음).

> **게이트는 상태가 `연기`인 것만 평가한다.** 설치를 끝낸 뒤 상태를 `연기→설치됨`으로 갱신하지 않으면, 다음 실행이 같은 결정을 또 시드한다.
